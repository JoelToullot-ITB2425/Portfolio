About the Project

In this project we managed to create an IT company from scratch, the objective of this IT company was to sell and buy PC components and upgrade them if possible, we managed to create a webserver using HTML, CSS and JS which had all the information of the company.

We build the webserver using LAMP and UbuntuServer 22.04, and we achieved connection in LAN using a private IP (for example https://192.168.1.1).

We managed our time with "sprint" which were 3 weeks phases in which the first days would talk about what would we do during that 3 weeks and who would do it, this was a very good way to manage our time and our workloads, we used proofhub to manage it online.

This was a simple yet good project!
